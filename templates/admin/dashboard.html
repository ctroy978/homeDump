Backup and restore

Back up the classroom data folder to a USB drive. Archives contain the database, attendance uploads, assignment PDFs, and claimed homework — not your .env password.

Stop the server before restoring. Restore moves your current data/ folder aside automatically, then unpacks the backup. 

# Backup to a USB drive (change the path to your drive)
uv run python scripts/backup_data.py /run/media/$USER/USB

# Restore from a backup archive
uv run python scripts/restore_data.py /run/media/$USER/USB/homedump-data-YYYYMMDD-HHMMSS.tar.gz --yes
