{% elif request.query_params.get('backup_error') == 'failed' %}

Download a compressed backup of classroom data to save on your computer, USB drive, or cloud storage. The file is a .tar.gz archive (similar to a zip file) containing your database, attendance uploads, assignment PDFs, and claimed homework.

Does not include your .env admin password — keep that somewhere safe separately.

Download backup

To restore later, stop the server and run the restore script on the classroom computer with your saved file: 

uv run python scripts/restore_data.py /path/to/homedump-data-YYYYMMDD-HHMMSS.tar.gz --yes
uv run main

You can also write backups directly to a USB drive: uv run python scripts/backup_data.py /run/media/$USER/USB
